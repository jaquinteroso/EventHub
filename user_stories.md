# EventHub - User Stories

### 1. Event Creation and Management
* **As an organizer**, I want to **create a new event with a title, description, and rich text formatting**, so that **I can provide all the necessary information to potential attendees.**
* **As an organizer**, I want to **save my event as a draft**, so that **I can finish editing it later before making it public.**
* **As an organizer**, I want to **set a maximum capacity for my event**, so that **I can manage the number of participants based on the venue's limits.**
* **As an organizer**, I want to **cancel an event I created**, so that **I can notify registered users if the activity cannot take place.**
* **As an organizer**, I want to **view a list of all registered attendees for my event**, so that **I can keep track of who is participating.**

### 2. Event Discovery and Browsing
* **As a user**, I want to **browse a list of all published events**, so that **I can find activities that interest me.**
* **As a user**, I want to **filter events by category (e.g., Sports, Study, Social)**, so that **I can quickly find specific types of gatherings.**
* **As a user**, I want to **view the details of an event including venue, available spots, and dates**, so that **I can decide whether to attend.**

### 3. Registration and Waiting List
* **As a participant**, I want to **register for an event**, so that **I can secure my spot and receive updates.**
* **As a participant**, I want to **be automatically placed on a waiting list if an event is full**, so that **I have a chance to attend if someone else cancels.**
* **As a participant**, I want to **be automatically promoted from the waiting list to confirmed status**, so that **I don't have to manually check for openings.**
* **As a participant**, I want to **cancel my registration**, so that **my spot can be released for someone else on the waiting list.**

### 4. Review and Rating
* **As an attendee**, I want to **rate a completed event (1-5 stars) and leave a comment**, so that **I can share my feedback and help others decide on future events.**
* **As a user**, I want to **read reviews of past events**, so that **I can know the quality of activities organized by certain members.**

### 5. Administrative Actions (Moderation)
* **As an administrator**, I want to **edit or remove any event on the platform**, so that **I can moderate inappropriate content or incorrect information.**
* **As an administrator**, I want to **manage venues and categories**, so that **the platform's metadata stays organized and up to date.**
* **As an administrator**, I want to **manage user accounts**, so that **I can ensure a safe environment for the university community.**
